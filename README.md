# Wookaao-Services

Live demo: https://shashankr6925.github.io/Wookaao-Services/

A fully responsive business landing page for **Wookaao Services**, designed to generate leads with clear CTAs and modern UI.

## Features
- Responsive layout (Desktop, Tablet, Mobile)
- Click-to-call buttons
- Email & WhatsApp integration
- Hero, Services, Features & Growth sections
- Sticky header with clickable logo
- Clean HTML & CSS (no frameworks)

## Tech Stack
- HTML5
- CSS3 (Flexbox & Grid)
- Font Awesome
